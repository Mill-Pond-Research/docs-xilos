# Xilos Documentation

Documentation site for [Xilos](https://www.xilos.ai) — intelligent agentic AI infrastructure platform by Mill Pond Research.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Tech Stack

- [Next.js](https://nextjs.org/) 14 (App Router)
- [Nextra](https://nextra.site/) 2 (docs theme)
- TypeScript
- Deployed on Vercel at [docs.xilos.ai](https://docs.xilos.ai)

## Structure

```
src/content/docs/
├── getting-started/     # Onboarding, quickstart, concepts
├── platform/            # Observe, Secure, Orchestrate pillars
├── integrations/        # Agent framework setup guides
├── api-reference/       # REST API documentation
├── deployment/          # SaaS, VPC, on-prem, fine-tuning
├── governance/          # Compliance, data residency, audit
├── guides/              # Step-by-step tutorials
└── architecture/        # System design docs
```

## Voice & Tone

- Precise, instructive, neutral, authoritative
- Active voice, present tense for capabilities
- Oxford commas
- Products capitalized: Xilos, WorkBench
- Pillars capitalized: Observe, Secure, Orchestrate
- Define acronyms on first use: LLM, SLM, PII, DLP
