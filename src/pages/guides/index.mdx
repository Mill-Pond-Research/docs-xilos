# Guides

Step-by-step guides for using Xilos features — from creating routing rules to fine-tuning models.

## Platform Guides

- **[Using the Prompt Playground](/guides/prompt-playground)** — Test prompts against different models with adjustable parameters.
- **[Create a Routing Rule](/guides/create-routing-rule)** — Route queries to specific models based on intent or keywords.
- **[Create a Restriction Rule](/guides/create-restriction-rule)** — Block, mask, or warn on queries matching specific patterns.
- **[Build a Multi-Model Workflow](/guides/build-workflow)** — Create multi-stage LLM pipelines with the Visual Workflow Builder.
- **[Enable Context Compression](/guides/enable-compression)** — Reduce token costs by 50-90% with three compression strategies.

## Integration Guides

- **[Set Up Webhooks](/guides/setup-webhooks)** — Receive event-driven notifications for query events.
- **[Set Up Routines](/guides/setup-routines)** — Schedule recurring agent queries with cron expressions.
- **[Set Up SIEM Export](/guides/setup-siem)** — Forward governance events to Splunk, Datadog, or other SIEM platforms.
- **[Manage Virtual Keys](/guides/virtual-keys)** — Create per-user API keys with budget and rate limits.

## Data and Fine-Tuning Guides

- **[Create Eval Datasets](/guides/eval-datasets)** — Create test datasets and run regression tests against routing rules.
- **[Generate Synthetic Data](/guides/synthetic-data)** — Use the SLM to generate edge-case test variants.
- **[Export Training Data](/guides/export-training-data)** — Export the query log as a filtered training dataset.
- **[Fine-Tune a Model](/guides/fine-tune-model)** — Fine-tune an open-source model using Xilos orchestration data.
