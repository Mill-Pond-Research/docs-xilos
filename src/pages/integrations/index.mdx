# Integrations

Xilos is OpenAI-compatible — if your tool speaks the OpenAI API, it works with Xilos. Connect your existing agents in minutes by changing the base URL and API key.

## OpenAI-Compatible API

All integrations use the same pattern:

1. Set the base URL to `https://api.xilos.ai/api/v1`
2. Set the API key to your Xilos API key
3. Use `"xilos"` as the model name for smart routing

That's it. Xilos handles routing, security, caching, and cost control automatically.

## Supported Frameworks

- **[Hermes Agent](/integrations/hermes)** — Connect Hermes Agent to Xilos via profile configuration.
- **[OpenClaw](/integrations/openclaw)** — Route OpenClaw traffic through Xilos.
- **[Claude Code](/integrations/claude-code)** — Point Claude Code at Xilos using environment variables.
- **[OpenAI SDK](/integrations/openai-sdk)** — Use the official OpenAI SDK with Xilos.
- **[LangChain](/integrations/langchain)** — Route LangChain agents through Xilos.
- **[GitHub Copilot](/integrations/copilot)** — Configure Copilot to route through Xilos.
- **[Kilo Code](/integrations/kilo-code)** — Connect Kilo Code for governance and cost control.
- **[OpenWebUI](/integrations/openwebui)** — Add Xilos as a connection in OpenWebUI.
- **[Custom Application](/integrations/custom-app)** — Integrate any application with the Xilos API.

> **Info:** Don't see your framework? Since Xilos is OpenAI-compatible, any tool that supports custom OpenAI base URLs will work. See [Custom Application](/integrations/custom-app) for generic integration steps.
