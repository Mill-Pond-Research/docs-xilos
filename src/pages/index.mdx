# Welcome to Xilos

Xilos is an intelligent agentic AI infrastructure platform by Mill Pond Research that observes, secures, and orchestrates agentic AI across enterprise networks. It operates as a middleware gateway that intercepts, analyzes, and processes every prompt before it reaches an LLM destination.

## The Three Pillars

Xilos is built on three foundational pillars:

## Key Differentiators

- **Semantic caching** — Reuses responses for semantically similar queries, even if phrased differently. Reduces cost and latency by 50-90%.
- **Query decomposition and contextual enrichment** — Breaks down complex queries and enriches them with relevant context from the Context Engine.
- **Comprehensive observability dashboards** — Real-time visibility into query volume, governance actions, cost trends, and orchestration metrics.
- **Auditable compliance logs** — Immutable audit trail capturing every governance action, restriction trigger, and security event.
- **Model-agnostic integration** — Unified API surface for OpenAI, Anthropic, Google, and privately hosted open-source models.
- **Intelligent routing** — Intent-based model routing using an internal Small Language Model (SLM) with a 16k token context window.
- **Three deployment options** — Managed SaaS, Private Cloud (VPC), and On-Premise (air-gapped capable).

## Get Started in 5 Minutes

Xilos is OpenAI-compatible — if your tool speaks the OpenAI API, it works with Xilos. Point your existing agents at Xilos and get governance, caching, and cost control immediately.

```bash
# Point any OpenAI-compatible client at Xilos
export OPENAI_API_KEY="your-xilos-api-key"
export OPENAI_BASE_URL="https://api.xilos.ai/api/v1"
```
